üîß Technical Details

Architecture

  • Backend: FastAPI with SQLAlchemy and PostgreSQL
  • Data Processing: Pandas for data manipulation and analysis
  • Visualization: Plotly for interactive charts and dashboards
  • Telegram API: Telethon for message collection
  • Rate Limiting: Custom rate limiter with configurable parameters

Data Schema

  • Message Metadata: ID, text, date, views, forwards, replies
  • Media Information: Type, filename, size, MIME type, caption
  • User Data: Creator username, first/last name
  • Channel Data: Username, forwarding information
  • Timestamps: Creation, update, and deletion tracking

Performance Features

  • Batch Processing: Efficient handling of large datasets
  • Incremental Updates: Smart collection that avoids duplicates
  • Memory Optimization: Pandas data cleaning and type optimization
  • Export Formats: Multiple output formats for different use cases

üìñ Usage Examples

Basic Collection

# Collect latest 100 messages from a channel python 0_getMediaFromTel.py --channel @SherwinVakiliLibrary --max-messages 100 # Collect older messages (incremental) python 0_getMediaFromTel.py --channel @SherwinVakiliLibrary --older --max-messages 100 # Save to JSON file python 0_getMediaFromTel.py --channel @SherwinVakiliLibrary --save-json-default

Data Export

# Export all messages to JSON python export_telegram_messages_pandas.py --format json --summary # Export to Excel with multiple sheets python export_telegram_messages_pandas.py --format excel --summary # Export specific channel data python export_telegram_messages_pandas.py --channel @SherwinVakiliLibrary

Dashboard Generation

# Generate default dashboard python analyze_telegram_data.py # Custom input/output files python analyze_telegram_data.py -i my_export.json -o my_dashboard.html # Verbose logging python analyze_telegram_data.py --verbose
